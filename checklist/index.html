ABOUT PROJECT

A small project for current work

A small project for a current work. The purpose of this HTML page is to check if a task was done properly. The list contains hot spots where the most mistakes are made. An employee must check all the boxes before closing the task in order to check himself. Reset button resets all the checkboxes for the next task.

HTML file contains CSS and JS code so it is easy to share and use for a number of employees.
